# 📸 Modern Image Gallery

A modern, responsive, and interactive Image Gallery built using **HTML, CSS, and JavaScript**. The project features category filtering, image lightbox preview, keyboard navigation, smooth animations, and a responsive design suitable for desktop, tablet, and mobile devices.

## 🚀 Features

### ✨ Responsive Design

* Fully responsive layout using CSS Grid.
* Works seamlessly on desktop, tablet, and mobile devices.

### 🖼️ Image Gallery

* Displays multiple high-quality sample images.
* Organized into categories:

  * Nature
  * Animals
  * Cities
  * Technology

### 🔍 Filter Functionality

* Filter images by category.
* Dynamic image display without page reload.

### 💡 Interactive Lightbox

* Click any image to open a fullscreen preview.
* Navigate images using:

  * Previous button
  * Next button
  * Keyboard arrow keys
* Close using:

  * Close button
  * ESC key

### 🎨 Modern UI Effects

* Smooth hover zoom effect.
* Overlay captions on hover.
* Fade-in animation for gallery items.
* Gradient header design.

### ⚡ Loading Animation

* Animated loading spinner displayed while page loads.
